This application has two purposes:

  1. Scrape craigslist for cars with good resale value and good annual milage
  2. Allow for custom queries that scrape craigslist and filter by good annual milage

By doing this, we hope consumers and resellers have a more efficient way to find cars with good value on craigslist.

This site was built with the Flask framework for Python, and uses Beautiful Soup for parsing.
